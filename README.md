# Deuk-Drive

개인용 클라우드 저장소 구축 프로젝트입니다.
Kotlin을 사용할 예정이고 서버는 Node.js를 사용하여 Android와 통신을 할 것입니다.
각 파일과 사진 문서등을 서버를 통해 업로드하고 어디서든지 내려받을 수 있도록 제작하는 것이 목표입니다.
